# DevOpsDockerProject  
This is my dockerized NodeJS/REACT app, using PostgreSQL, REDIS, and NGINX  
## To run this app simply run command:  
```
docker-compose up --build
```
### To work properly add .env file with values:  
```
REDIS_HOST=…  
REDIS_PORT=…  
PGUSER=…  
PGHOST=…  
PGDATABASE=…  
PGPASSWORD=…  
PGPORT=…  
POSTGRES_USER=…  
POSTGRES_HOST=…  
POSTGRES_DATABASE=…  
POSTGRES_PASSWORD=…  
POSTGRES_PORT=…  
```
